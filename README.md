# VD-TriggerEvent

A lightweight FiveM development tool that provides an in-game interface for triggering client and server events for debugging. Includes an NUI UI, command. Forget about RedEngine & Macho Injectors.

---

## Features

• NUI interface for sending events  
• Trigger client or server events  
• /eventtester command to open UI  
• Toggle UI open/close  
• Designed for rapid debugging during development  

---

## Installation

1. Place "vd-triggerevent" into your server’s resources folder.  
2. Add this line to your server.cfg:

ensure vd-triggerevent

---

## Usage

Open the tester:
• Use the command: /eventtester

Close the tester: 
• click the close button in the UI
